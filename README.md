

## About

###Pre-Requisites:

*npm
*bower
*grunt

####Clone

	https://github.com/jvrunion/angular-backbase-generator.git

	$ cd backbase-angular
	$ npm install
	$ bower install

Than run tests:
	
	$ grunt test

Serve the development state of project

	$ grunt serve

Or build project and execute index.html file from dist directory

	$ grunt build
